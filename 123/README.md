# 游戏学习提醒

一个 Java 桌面程序，在你打开游戏时弹出提醒，督促先完成每日学习任务。

## 功能

- 后台监控系统进程，检测到指定游戏启动时弹出提醒
- 支持自定义添加任意游戏进程名
- 学习链接支持网址和本地路径（视频文件、文件夹等）
- 系统托盘常驻运行，不占任务栏空间
- 每天同一游戏只提醒一次

## 使用方法

### 1. 启动

双击 `run.bat` 即可编译并运行（需要 JDK 8+）。

### 2. 设置

首次启动会自动弹出设置窗口，也可通过右键/双击托盘图标打开：

- **添加游戏进程**：输入游戏进程名，如 `VALORANT-Win64-Shipping`、`GenshinImpact`（不区分大小写，可省略 `.exe`）
- **设置学习链接**：输入网址或点击「浏览」选择本地文件/文件夹

### 3. 运行

设置保存后，程序自动开始在后台监控。检测到游戏进程时弹出提醒窗口：

- **去学习** → 打开你设置的学习链接
- **跳过** → 关闭提醒

## 配置存储

配置文件保存在 `C:\Users\你的用户名\.game_reminder_config.json`。

## 如何获取游戏进程名

1. 启动游戏
2. 按 `Ctrl+Shift+Esc` 打开任务管理器
3. 在「详细信息」标签页中找到游戏进程名（如 `GenshinImpact.exe`）
4. 将进程名（不含 `.exe` 也可以）添加到设置中

## 技术栈

- Java 8+
- Swing（UI）
- AWT SystemTray（系统托盘）
- 无外部依赖
